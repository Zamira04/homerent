# 🏠 HomeRent

Modern rental website with booking system.

## 🌐 Live Demo
https://zamira04.github.io/homerent/

## 📌 Features
- 房源列表 (Property catalog)
- 房源详情页面 (Details page)
- 在线预订系统 (Booking system)
- 自动价格计算 (Price calculation)
- 响应式设计 (Responsive design)

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
